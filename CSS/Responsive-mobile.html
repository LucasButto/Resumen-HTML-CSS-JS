Responsive Design - Mobile First

Mobile first, una forma de mejora progresiva, es un enfoque de desarrollo y diseño web que se enfoca en la priorización del diseño y el desarrollo para dispositivos móviles por encima del diseño y desarrollo para pantallas de escritorio. ACLARACIÓN: Solo se utilizan los conocimientos del resumen adquiridos con los enlaces previos, con mas detalles se puede mejorar y simplificar
