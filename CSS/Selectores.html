Selectores

  • * (universal): Selector que aplica a todos los elementos.
  • Selector de tipo: Los que mencionamos al elemento directamente (hx, p, ul, table, etc)
  • .NombreDeLaClase (Selector por clases): Modifica todos los elementos que contienen esa clase

    • Sin clase

    Con clase

    Sin clase

  • #NombreDelID Selector por ID: Modifica solo los elementos que contienen ese identificador, a diferencia de la clase los id deben ser unicos
  • [NombreDelAtributo="ValorDelAtributo"] Selector por atributo: Modifica todos los elementos que contienen tanto el nombre como el valor del atributo
  • Selector por descendencia: se aplica a los elementos que están dentro de otro elemento

  • Selector por pseudoclases: Una pseudoclase CSS es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado. Por ejemplo, :hover aplicará un estilo cuando el usuario haga hover sobre el elemento especificado por el selector.
